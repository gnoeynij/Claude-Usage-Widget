# v2.3.0 — 중복 실행 방지 + Claude Fable 5 단가 + 알림 토스트 중복 fix

위젯이 항상 하나만 뜨도록 막고, 새로 출시된 Claude Fable 5의 비용을 정확히 계산하며, 임계치 알림이 한 번에 두 개씩 뜨던 문제를 고쳤습니다.

## 새 기능

- **단일 인스턴스 락** — 위젯이 이미 실행 중일 때 다시 실행하면, 새 창을 띄우는 대신 떠 있던 창을 앞으로 끌어올립니다(트레이에 숨겨 둔 상태였다면 다시 표시). 중복 프로세스가 생기지 않습니다.
- **Claude Fable 5 단가 지원** — 2026-06-09 출시된 Fable 5(Opus 위의 새 티어, $10/$50 per MTok)의 비용을 공식 단가표대로 계산하고, Detail 모델 카드에 전용 색으로 표시합니다.

## 버그 수정

- **알림 토스트 중복** — 사용량이 100%인 상태로 동기화될 때 85%·95% 임계치를 한 번에 넘으면서 토스트가 두 개 동시에 뜨던 문제를 고쳤습니다. 이제 가장 높은 한 단계만 알립니다.

## 비고

- 모든 비용은 로컬 JSONL × 공식 단가표 기반 *추정치*입니다(구독 사용자는 실제 토큰당 과금 없음).
- Fable 5는 2026-06-22까지 Pro/Max/Team 플랜에 무료 포함, 6-23부터 사용 크레딧 차감으로 전환됩니다(Anthropic 정책 — 컴퓨트 수용량 확보 시 복원 예정).
