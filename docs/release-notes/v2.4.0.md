# v2.4.0 — Limit projection + Detail redesign + Widget guide

Real-time "on pace to hit the limit" warnings across all modes, a fully redesigned Detail screen centered on a daily cost chart, and an interactive guide you can open from Settings.

## What's new

### Limit projection

- **Ghost marker** — a translucent arc / dot overlaid on the Normal and Mini donut/bar shows where usage is expected to land at reset if the current pace continues.
- **⚠ Risk badge (Mini)** — appears only when you're on pace to hit a limit. Click it to see a per-limit breakdown with current % and estimated time to cap.
- **Tray notification** — "at this pace you'll hit the limit before reset" fires in the background even when the widget is hidden.
- **Instant-pace blend** — uses the larger of the rolling average and a short EMA, so a sudden burst is flagged quickly.
- **Merged caption (Normal)** — session reset time and projection now fit on one line, saving vertical space.

### Detail mode redesign

- **Daily cost chart** — stacked model-mix bar chart is now the hero. Switch between 7 / 14 / 30-day ranges, click any bar to inspect that day, see avg / trend (↗↘%) / peak in a summary row.
- **Device-toggle history** — when a sync folder is configured, a toggle in the chart card switches between this device and the fleet-wide sum.
- **Unused models shown** — Opus · Fable · Sonnet · Haiku are always listed; models with no activity in the window show $0.
- **Totals card** — left column: this week / this month; right column: lifetime this device / all devices.

### Widget guide

- **Settings → Guide button** — opens a separate window rendering the real widget.
- **Usage slider** — drag to scrub through safe and risk states and watch the UI respond.
- **Measured callouts** — Mini · Normal · Detail · Settings tabs, each with connector lines to live anchors.

## Notes

- All costs are *estimates* derived from local JSONL × official pricing (subscribers are not billed per-token).

## 새 기능

### 한도 투영 (Limit Projection)

- **고스트 마커** — Normal·Mini 모드 도넛/막대에 반투명 아크·도트가 겹쳐 표시됩니다. 현재 페이스가 유지될 때 reset 시점의 예상 사용량을 나타냅니다.
- **⚠ 위험 배지 (Mini)** — 한도 도달이 예상될 때만 우상단에 ⚠ 배지가 나타납니다. 클릭하면 세션·주간별 현재 % + 예상 시간을 바로 확인할 수 있습니다.
- **트레이 예측 알림** — "이 속도라면 reset 전에 한도에 도달합니다"가 배경에서도 알려줍니다.
- **순간-페이스 블렌드** — 갑작스러운 사용 폭주를 빠르게 반영하기 위해 전체 평균과 최근 EMA 중 큰 값을 사용합니다.
- **캡션 한 줄 병합 (Normal)** — 세션 reset 시간과 투영 결과를 한 줄로 합쳐 세로 공간을 절약합니다.

### Detail 모드 전면 재설계

- **일별 비용 차트** — 모델 믹스 누적 막대 차트가 메인으로 올라옵니다. 7·14·30일 범위 전환, 막대 클릭으로 날짜 선택, 피크/평균/추세(↗↘%) 요약 행을 제공합니다.
- **기기 통합 히스토리** — 클라우드 폴더가 설정된 경우, 차트 우측에 기기 토글이 나타나 이 기기 / 전체 기기 합산을 바로 전환할 수 있습니다.
- **미사용 모델도 표시** — Opus·Fable·Sonnet·Haiku 4개 모델이 기간 중 사용 여부와 관계없이 항상 표시됩니다. 미사용 모델은 $0으로 표기됩니다.
- **Totals 카드 재정렬** — 좌측에 이번 주·이번 달, 우측에 이 기기·전체 누적을 나란히 배치합니다.

### 위젯 가이드

- **Settings → 가이드 버튼** — 별도 창에서 실제 위젯이 그대로 렌더링됩니다.
- **사용량 슬라이더** — 드래그하면 위험·안전 경계를 직접 넘나들며 각 상태의 UI 변화를 확인할 수 있습니다.
- **측정 기반 callout** — Mini·Normal·Detail·Settings 4개 탭, 각 UI 요소에 연결선과 설명이 붙습니다.
