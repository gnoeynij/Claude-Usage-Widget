# v2.2.0 — 누적 비용(기기별·기기 통합) + 실시간 카운트다운 + 인앱 알림

사용 비용을 "줄지 않는 누적"으로 보고, 여러 PC의 비용을 하나로 합치고, 알림을 위젯에 맞는 디자인으로 바꿨습니다. 세션 남은 시간도 실시간으로 똑딱입니다.

## 새 기능

- **누적 비용 (이 기기)** — Detail 화면 통계에 "누적 · 이 기기"가 생겼습니다. 기존 비용은 *현재 디스크에 남은 로그* 기준이라 오래된 세션이 정리되면 줄어들 수 있는데, 누적 비용은 한 번 집계한 값을 계속 유지합니다(줄어들지 않음).
- **누적 비용 (전체 기기)** — 설정의 "기기 통합 누적"에서 여러 PC가 공유하는 클라우드 동기화 폴더(OneDrive·iCloud·Dropbox·Google Drive 등)를 지정하면, 각 기기의 누적 비용을 자동으로 합산해 "누적 · 전체 기기"로 보여줍니다(연결된 기기 수는 설정 화면에 표시). 별도 서버 없이 각 기기가 그 폴더에 자기 파일만 기록합니다.
- **실시간 세션 카운트다운** — 세션 리셋과 활성 5시간 블록의 남은 시간이 sync를 기다리지 않고 초 단위로 실시간 갱신됩니다.
- **인앱 Liquid Glass 알림** — 사용량 임계치(85%/95%) 알림이, 위젯이 화면에 보일 때는 위젯 디자인과 어울리는 인앱 토스트로 뜹니다(트레이로 숨겼을 때만 기존 OS 알림). Windows에서 알림이 "PowerShell"로 표시되던 것도 정식 설치 시 해소됩니다.
- **Detail 인사이트** — 기간별 카드에 "월말 예상 비용"(현재 페이스 기준 추정), 모델 카드에 "모델 믹스 %"(가족별 비용 점유율)가 추가됐습니다.

## 버그 수정

- 신규 `extra_usage`(추가 크레딧) API 필드가 `null`로 올 때 사용량 동기화 전체가 깨지던 문제를 고쳤습니다.

## 내부 개선

- 미사용 코드·플러그인·CSS 토큰 정리, 비용 계산(`resolve`) 캐시로 헤비 사용자의 Detail 동기화 부담을 줄였습니다.

## 비고

- 모든 비용은 로컬 JSONL × 공식 단가표 기반 *추정치*입니다(구독 사용자는 실제 토큰당 과금 없음).
- 기기 통합은 클라우드 동기화 지연만큼 늦게 반영되며, 데이터는 본인 클라우드에만 저장됩니다.

## 다운로드

### Windows
- `Claude Widget_2.2.0_x64-setup.exe` — NSIS 인스톨러 (권장)
- v2.0.x / v2.1.x 사용자는 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.2.0_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`
