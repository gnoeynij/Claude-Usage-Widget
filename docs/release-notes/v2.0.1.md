# v2.0.1 — 첫 v2.0.x 공개 릴리즈

v2.0.0 ([release notes](v2.0.0.md)) 은 internal cut 으로 GitHub Releases 에는 발행되지 않았음. **v2.0.1 이 v1.5.1 (PyQt6) 이후 첫 공개 stable 릴리즈** — 기능 집합은 v2.0.0 과 동일.

## v2.0.0 → v2.0.1 변경

- **서명 키 교체** — 새 minisign keypair 발급에 따른 `tauri.conf.json` `plugins.updater.pubkey` 갱신. v2.0.0 internal 빌드를 받았던 사용자는 자동 업데이트 불가 (수동 재설치 필요) — 외부 사용자 영향 없음 (v2.0.0 미공개).
- v2.0.x 위젯 스크린샷 3장 (normal / detail / mini) — README 자산 정비.

## v2.0.0 에서 이어진 주요 기능 (재공지)

- 스택: PyQt6 → Tauri 2 (Rust + System WebView) + SolidJS + TS + UnoCSS
- 디자인: Liquid Glass + Win11 Mica/Acrylic, `tokens.css` 단일 출처
- 3-mode 뷰 (Mini / Normal / Detail) + SegmentedControl·트레이 즉시 전환
- 자동 업데이트 (tauri-plugin-updater) — 이 릴리즈부터 실제 동작
- 시스템 트레이 좌클릭 toggle, X 버튼 hide → 트레이로 복귀
- OAuth 토큰: `.credentials.json` mtime polling + 만료 시 자동 retry
- 한국어 / 영어 즉시 전환, `widget.log` 진단 로그

상세는 [v2.0.0.md](v2.0.0.md) 참조.
