# v2.1.1 — 배경 투명도 슬라이더 보완

배경 투명도 100% 에서 위젯이 완전히 사라지고, 라이트 모드 Settings 오버레이가 자연스러운 흰 wash 로 paint 된다.

## 알려진 이슈

v2.1.0 그대로:
- OAuth 토큰 자동 refresh 미구현 — `.credentials.json` 만료 시 `claude` CLI 1회 실행으로 갱신 필요 ([BACKLOG](../../BACKLOG.md) P1).
- macOS Universal binary 미지원 — Apple Silicon only ([BACKLOG](../../BACKLOG.md) P2).
- macOS 트레이 템플릿 이미지 미적용 — 다크 메뉴바 자동 색반전 X ([BACKLOG](../../BACKLOG.md) P2).

## 다운로드

### Windows
- `Claude Widget_2.1.1_x64-setup.exe` — NSIS 인스톨러 (권장)
- v2.0.x / v2.1.0 사용자는 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.1.1_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

- macOS 11 Big Sur 이상 (Apple Silicon)
