# v2.1.0 — 사용량 알림 + Detail 모드 성능

위젯이 *passive 표시*에서 *active 알림*으로. Detail 모드 latency 도 mtime 캐싱으로 단축.

## 새 기능

- **사용량 임계치 OS 알림** — 5시간 세션 / 7일 주간 한도 사용량이 85% / 95% 에 도달하면 OS native notification 발송. 같은 블록 안에서 각 임계치당 1회. 권한은 첫 임계치 도달 시점에 요청.
- **OS 다크 모드 자동 따라가기** — Settings 에서 명시 토글하기 전까지 `prefers-color-scheme` 추종.
- **Donut 클릭 = 지금 동기화** — Normal / Mini / Detail 3 모드 모두 같은 동작.
- **커스텀 스크롤바** — Liquid Glass 패널과 톤 맞춤 (8px 폭 + 투명 트랙).
- **로그 회전 정책** — `widget.log` 누적 디스크 사용 ~5MB 상한.

## 변경

- **Detail 모드 mtime 캐싱** — Detail 모드 진입 / sync 시 변경된 파일만 재파싱. 헤비 사용자 (수 GB 누적) 환경에서 첫 진입 시간 단축.
- **버전 헬퍼 스크립트** — 6 파일 버전 한 줄 일괄 bump (`node scripts/bump-version.mjs 2.1.0`).

## 알려진 이슈

없음.

## 다운로드

### Windows
- `Claude Widget_2.1.0_x64-setup.exe` — NSIS 인스톨러 (권장)
- 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.1.0_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

## 시스템 요구사항

- Windows 10/11 (Win11 권장 — Mica vibrancy)
- macOS 11 Big Sur 이상 (Apple Silicon)
- WebView2 Runtime (Win10 인스톨러가 자동 설치)
