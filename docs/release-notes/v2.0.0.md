# v2.0.0 — Liquid Glass Rewrite

PyQt6 → Tauri 2 + SolidJS + Vite 전면 재작성. visionOS·Liquid Glass 디자인 언어로 처음부터 다시.

## 새 기능

### 3-mode 위젯
- **Mini** (240×112) — Donut + 2-row capsule. 항상 위 / 최소 공간
- **Normal** (360×420) — 기존 기본 모드 + 깔끔 정리
- **Detail** (600×680) — 기간별 비용·최근 블록·모델별 사용량·통계 4-카드 대시보드

푸터 SegmentedControl 또는 트레이 메뉴로 즉시 전환. 모드별 default size + minSize 자동 적용 + 사용자가 조정한 크기는 모드별로 기억.

### Liquid Glass + Win11 Mica/Acrylic
시스템 backdrop과 OS-level vibrancy 합성. 배경 투명도 슬라이더는 *배경만 fade* (텍스트·donut·게이지는 항상 불투명).

### 자동 업데이트
부팅 3초 후 silent check + Settings 수동 버튼 + 발견 시 톱니바퀴 점 뱃지 + 백그라운드 다운로드 + "지금 재시작" 버튼.

### 트레이/태스크바 아이콘 v2
- Anthropic 픽셀 캐릭터 + radial halo (Gaussian 5-stop falloff)
- threshold 색: 녹/주/적 (Apple stoplight)
- 호흡 효과 (4초 cycle, alpha 37%~92%) · Settings에서 toggle
- 1px 검은 stroke로 모든 배경에서 contrast 보장
- 오류 시 회색 halo + 우상단 빨간 status dot

### OAuth 토큰 자동 회복
- `.credentials.json` mtime polling으로 Claude Code CLI 갱신 자동 감지
- 만료 시 사용자에게 i18n 알림 + 갱신 후 자동 retry
- auto-sync default 5분
