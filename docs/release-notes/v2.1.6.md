# v2.1.6 — 투명도 즉시 반영 수정 (macOS)

macOS에서 배경 투명도를 조절하거나 위젯을 새로 켰을 때, 투명도가 바로 반영되지 않고 창 모드(Mini/Normal/Detail)를 한 번 바꿔야 적용되던 문제를 고쳤습니다. 이제 슬라이더를 움직이거나 위젯을 켜면 투명도가 곧바로 보입니다.

> v2.1.5에서 투명도를 50%처럼 설정했는데도 불투명하게 보이던 분들은, 이 업데이트 후 정상적으로 반영됩니다.

없음.

## 다운로드

### Windows
- `Claude Widget_2.1.6_x64-setup.exe` — NSIS 인스톨러 (권장)
- v2.0.x / v2.1.x 사용자는 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.1.6_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

## 시스템 요구사항

- Windows 10/11 (Win11 권장 — Mica vibrancy)
- macOS 11 Big Sur 이상 (Apple Silicon)
- WebView2 Runtime (Win10 인스톨러가 자동 설치)
