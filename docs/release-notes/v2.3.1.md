# v2.3.1 — 비용 누락 방지(누적·일별 히스토리) + 알림/항상위 버그 fix + fast 단가

여러 버그를 고치고, 비용 기록이 사라지지 않도록 보강했습니다. JSONL이 정리돼도 일별 비용 내역이 남고, 항상-위·알림 동작도 정상화됩니다.

## 새 기능

- **일별 비용 히스토리 (정리돼도 보존)** — Detail 화면 하단에 날짜별 비용이 모델 믹스 막대와 함께 쌓입니다. Claude Code가 오래된 로그(JSONL)를 자동 정리해도, 그 전에 매 sync마다 일별×모델 비용을 영구 저장해 두므로 내역이 사라지지 않습니다.
- **모든 모드에서 누적 비용 적립** — 기존엔 Detail 화면을 열어둘 때만 "줄지 않는 누적 비용"이 갱신됐는데, 이제 Mini/Normal 모드에 있어도 매 sync마다 적립됩니다. 로그가 정리되기 전에 비용을 잡아 누적 정확도가 올라갑니다.
- **fast 모드 단가 + Mythos 5** — `/fast`(speed: fast) 사용 시 Opus 단가를 정확히(4.8 2배·4.6/4.7 6배) 반영하고, Mythos 5 모델 단가를 추가했습니다.

## 버그 수정

- **알림 토스트가 위젯 콘텐츠를 밀어내던 문제** — 임계치 알림이 헤더·도넛을 아래로 밀어내지 않고 위에 겹쳐 표시되도록 고쳤습니다(Portal 오버레이).
- **알림 토스트 중복** — 100% 상태로 동기화될 때 85%·95% 토스트가 동시에 두 개 뜨던 문제를 고쳤습니다(가장 높은 단계만 알림).
- **재시작 시 "항상 위"가 풀리던 문제** — 위젯을 껐다 켜면 항상-위 설정을 다시 켜야 했는데, 부팅 시 자동으로 적용되도록 고쳤습니다.

## 비고

- 모든 비용은 로컬 JSONL × 공식 단가표 기반 *추정치*입니다(구독 사용자는 실제 토큰당 과금 없음).
- 일별 히스토리·누적은 기기별 로컬 기록입니다(클라우드 통합은 후속 예정).
