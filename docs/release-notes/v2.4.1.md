# v2.4.1 — Guide polish & projection fixes

A maintenance release that sharpens the limit-projection warnings and the
in-app guide from v2.4.0, with assorted layout and stability fixes.

## Fixed

- **Mini — double-click no longer maximizes** the window. Expand with the
  bottom handle (or the footer modes); the tooltip and guide now say so.
- **Projection** — the "on pace to exceed the limit" warning is no longer
  hidden when the projected value rounds to 99.x%; weekly projection starts
  warning slightly earlier in the week.
- **Always-on-top** is no longer dropped every time the window resizes.
- **Guide** — fixed weekly-caption clipping in Normal, re-enabled the Mini ⚠
  badge click, corrected callout positions, and the ÷ in the projection
  formula now renders correctly at small font sizes.

## Changed

- **Guide window** is now fixed-size with a cleaner aligned layout and an
  inline explanation of how projected usage is estimated.
- Footer and guide header use a centered grid layout.

## Notes

- All costs are *estimates* derived from local JSONL × official pricing
  (subscribers are not billed per-token).

## Download

### Windows
- `Claude Widget_2.4.1_x64-setup.exe` — NSIS installer (recommended)
- v2.0.x and later: Settings → "Check for updates" for automatic delivery

### macOS (Apple Silicon)
- `Claude Widget_2.4.1_aarch64.dmg` — DMG installer
- First-launch Gatekeeper bypass: right-click → Open, or
  `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

## System requirements

- Windows 10/11 (Win11 recommended — Mica vibrancy)
- macOS 11 Big Sur or later (Apple Silicon)
- WebView2 Runtime (auto-installed by the Win10 installer)
