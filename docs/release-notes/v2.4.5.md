# v2.4.5 — Claude Sonnet 5 pricing

A small release that adds cost tracking for the newly released Claude Sonnet 5
model, plus a minor Detail-view chart label fix.

## Fixed

- **Claude Sonnet 5 usage now shows a real cost instead of $0.** Anthropic
  released Sonnet 5 (`claude-sonnet-5`) on June 30, 2026; this widget didn't
  recognize the model id yet, so any Sonnet 5 usage was silently priced at
  $0. Now priced at Anthropic's introductory rate ($2 / $10 per MTok,
  in effect through August 31, 2026 — this widget will need a follow-up
  update when the standard $3 / $15 rate takes effect).
- **Detail view — removed a redundant max-value label** that overlapped the
  peak bar's own label in the daily cost chart.

## Notes

- All costs are *estimates* derived from local JSONL × official pricing
  (subscribers are not billed per-token).

## Download

### Windows
- `Claude Widget_2.4.5_x64-setup.exe` — NSIS installer (recommended)
- v2.0.x and later: Settings → "Check for updates" for automatic delivery

### macOS (Apple Silicon)
- `Claude Widget_2.4.5_aarch64.dmg` — DMG installer
- First-launch Gatekeeper bypass: right-click → Open, or
  `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

## System requirements

- Windows 10/11 (Win11 recommended — Mica vibrancy)
- macOS 11 Big Sur or later (Apple Silicon)
- WebView2 Runtime (auto-installed by the Win10 installer)
