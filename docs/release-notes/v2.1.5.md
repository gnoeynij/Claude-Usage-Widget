# v2.1.5 — 비용 계산 정확도 개선

사용량 비용(Detail 모드) 계산을 Anthropic 공식 단가표에 맞춰 정정했습니다. 그동안 일부 모델의 비용이 부정확하게 표시되던 문제를 바로잡은 것이 이번 릴리즈의 핵심입니다.

## 주요 변경

### 비용 정확도 (중요)
- **Opus 4.5 / 4.6 / 4.7 단가 정정** — 기존에 백만 토큰당 $15/$75(입력/출력)로 **3배 과대** 계산되던 것을 실제 단가 $5/$25로 바로잡았습니다. Opus를 많이 사용하셨다면 표시 비용이 이전의 약 1/3 수준으로 내려갑니다.
- **1시간 캐시 단가 분리** — 5분/1시간 prompt cache 단가를 분리해 계산합니다. 1시간 캐시를 쓰는 경우 그동안 과소 계산되던 부분이 정확해집니다.
- **웹 검색 · 미국 추론 비용 반영** — 서버 측 웹 검색($10/1,000건)과 `inference_geo: us`(토큰 비용 1.1배)를 비용에 포함합니다.
- **단종 예정 모델 단가 유지** — Opus 4/4.1, Sonnet 4 등 deprecated 모델도 비용이 $0으로 누락되지 않도록 단가를 유지합니다.

### 기타
- 업데이트 확인 시 "최신 버전입니다" 안내에 현재 버전 번호를 함께 표시합니다.
- 내부: 비용·집계 로직 단위 테스트 20개 추가 + CI에서 Windows·macOS 자동 검증(회귀 방어).
