# v2.1.5 — 비용 계산 정확도 개선

사용량 비용(Detail 화면)을 Anthropic 공식 가격에 맞춰 정정했습니다. 특히 Opus 모델 사용 비용이 그동안 실제보다 약 3배 높게 표시되던 문제를 바로잡았습니다 — Opus를 자주 쓰셨다면 표시 비용이 이전의 약 1/3 수준으로 내려갑니다.

함께 개선된 점:
- 캐시 기능(반복되는 입력을 재사용해 비용을 아끼는 기능)을 쓸 때의 비용 계산이 더 정확해졌습니다.
- 웹 검색 등 추가 요금이 발생하는 사용분도 비용에 반영합니다.
- 곧 지원 종료되는 구형 모델(Opus 4/4.1, Sonnet 4 등)의 비용도 0으로 빠지지 않고 정상 계산됩니다.
- 업데이트 확인 화면에 현재 버전 번호를 함께 표시합니다.
- 투명도를 설정해 둔 경우 첫 실행(특히 업데이트 직후)부터 바로 적용됩니다 — 이전에는 모드를 한 번 바꿔야 투명도가 적용되던 문제를 고쳤습니다.

## 다운로드

### Windows
- `Claude Widget_2.1.5_x64-setup.exe` — NSIS 인스톨러 (권장)
- v2.0.x / v2.1.x 사용자는 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.1.5_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`
