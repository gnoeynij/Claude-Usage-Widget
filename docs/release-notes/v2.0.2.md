# v2.0.2 — macOS 첫 정식 릴리즈

**Windows + macOS 양 플랫폼 자동 업데이트 동시 활성화.** v2.0.x 라인의 macOS 지원 시작점.

## 새 기능

### macOS 지원 (PR #1, PR #2)

- **Vibrancy** — `NSVisualEffectMaterial.HudWindow` 적용 ([`vibrancy_mac.rs`](../../src-tauri/src/vibrancy_mac.rs)). Win11 Mica 와 시각적 정합.
- **Keychain credentials** — macOS Claude Code CLI 가 토큰을 `~/.claude/.credentials.json` 대신 *login Keychain* (`Claude Code-credentials` 서비스, `$USER` 계정) 에 저장. Rust `keyring` crate 매치 실패로 `security` CLI subprocess 호출로 우회 ([`usage_api.rs`](../../src-tauri/src/usage_api.rs)).
- **Drag region** — WKWebView 가 `-webkit-app-region: drag` 미지원. `getCurrentWindow().startDragging()` JS API + capability `core:window:allow-start-dragging` 로 fallback.
- **DMG 번들** — Apple Silicon 전용 (Intel Mac 미지원, [BACKLOG.md](../../BACKLOG.md) P2).
- **Ad-hoc 서명** — Apple Developer Program 미가입. 첫 실행 시 Gatekeeper 경고 → 우클릭 → 열기 또는 `xattr -d com.apple.quarantine`.

### macOS 검은 모서리 fix (PR #2)

NSVisualEffectView 가 contentView 의 `masksToBounds` 로 clip 되지 않는 회귀. NSWindow `opaque=false` + `backgroundColor=clearColor` 명시 + 모든 직접 subview layer 에 cornerRadius mask 양쪽 다 적용.

### 자동 업데이트 — 양 OS 통합

- `scripts/make-updater-manifest.mjs` 가 Windows NSIS `.sig` + macOS `.app.tar.gz.sig` 둘 다 자동 감지해 `latest.json` 의 platforms 키를 합본.
- minisign 키 (Apple 과 무관) 로 양 OS 동일 서명.

### UX 개선

- **Detail 모드 Active 시간당 비용** — 현재 5h 블록 분당 cost 환산.
- **Models 토큰 열 추가** — 모델별 input / output / cache 토큰 분해.
- **Normal/Detail drag overlay** — 빈 영역 드래그 가능 여부 시각 힌트.
- **`formatTokens`** — 1.2K / 3.4M 단위 일관 표기.
- **트레이 전용 정사각 PNG** — 작은 트레이 영역에서 시각적으로 키움.
- **`startWindowDrag` 공통화** — `src/utils/drag.ts` 로 분리, OS 분기 단일화.
