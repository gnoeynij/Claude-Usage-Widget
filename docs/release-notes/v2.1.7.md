# v2.1.7 — Opus 4.8 비용 정정 + 구독 플랜 표시

5월 28일 출시된 Claude Opus 4.8의 사용 비용이 실제의 3배로 부풀려져 표시되던 문제를 고쳤습니다. 또한 설정 패널에 현재 구독 플랜(예: Max 20×)을 작은 칩으로 보여줍니다.

## 변경 사항

- **Opus 4.8 비용 정정** — Opus 4.8 단가가 위젯에 등록되지 않아 구버전 Opus 단가($15/$75)로 잘못 계산되어, Opus 4.8 사용분의 비용이 3배로 과대 표시되던 문제를 고쳤습니다. 이제 공식 단가($5/$25)로 정확히 집계됩니다.
- **구독 플랜 표시** — 설정 패널 상단 "설정" 타이틀 옆에 현재 구독 플랜(Max 20×, Pro 등)을 칩으로 표시합니다. 로그인하지 않은 상태에서는 표시되지 않습니다.
