# v2.0.3 — 설정 보존 안정화 + 트레이 i18n + 상태 알림 확장

v2.0.x 라인의 사용자 설정 보존을 안정화하고, 일상에서 자주 마주치는 트레이·알림 표면을 다듬은 정비 릴리즈.

## 새 기능

### 트레이 메뉴 한국어 지원
시스템 트레이 우클릭 메뉴 (Show/Hide · Mini/Normal/Detail 모드 · 지금 동기화 · 종료) 가 위젯 언어 설정에 따라 한/영 전환. 기존엔 언어 설정과 무관하게 영어 고정.

### 상태 알림 4종
사용량 동기화 실패 시 위젯 상단에 *어떤 상태*인지 안내 배너 표시. 기존엔 *토큰 만료* 하나만 보였고 나머지는 silent fail.

- **인증 토큰 만료** — `claude` CLI 1회 실행으로 자동 복구 안내
- **인증 정보 없음** — `claude` CLI 로 로그인 안내 (`.credentials.json` / Keychain 미발견)
- **요청 제한** — Anthropic API 가 잠시 throttling 중, 다음 주기 자동 재시도
- **네트워크 오류** — Anthropic API 에 연결 불가, 네트워크 상태 점검 안내

각 상태별 lucide 아이콘 + tone (경고/정보/위험) 색상.

## 변경

### 설정 영구 저장
**언어 · 다크 모드 · 배경 투명도 · 동기화 주기 · 항상 위 · 모드 6 항목** 이 위젯 재시작 후에도 유지. v2.0.0~v2.0.2 에서는 변경 사항이 *메모리만 갱신*되고 디스크에 저장 안 돼, 매 재시작마다 기본값 (영어 · 라이트 · 0% · 5분 · 끔 · Normal) 으로 복원되던 회귀 해소. ([회귀 사례 §15](../../CLAUDE.md))

### PyQt6 마이그레이션 정렬
v1.5.x → v2.0.x 업그레이드 시 기존 설정 이관 코드의 저장 위치가 새 위치와 어긋나 *마이그레이션이 실행되어도 적용 안 되던* 영역 정렬. v1.5.x 의 동기화 주기 단위 (초) → v2.0.x (분) 변환 포함.

> 본인 머신에 v2.0.x 가 이미 설치된 사용자는 영향 없음. v1.5.x 에서 직접 업그레이드하는 사용자만 효과.

## 알려진 빈 구멍

- OAuth 토큰 자동 refresh 미구현 — `.credentials.json` 만료 시 `claude` CLI 1회 실행으로 갱신 필요 ([BACKLOG](../../BACKLOG.md) P1).
- macOS Universal binary 미지원 — Apple Silicon only (Intel Mac 수요 시 후속, BACKLOG P2).
- macOS 트레이 템플릿 이미지 미적용 — 다크 메뉴바 자동 색반전 X (BACKLOG P2).
