# v2.1.4 — 트레이 아이콘 + macOS 모서리 다듬기

시스템 트레이 아이콘이 sync 상태(정상/실패)를 즉시 반영합니다.
트레이에 표시되는 아이콘이 다른 시스템 아이콘들과 비슷한 크기로 보이도록 개선했고, macOS 위젯 모서리도 좀 더 둥글게(24px) 다듬었습니다.

## 알려진 이슈

없음.

## 다운로드

### Windows
- `Claude Widget_2.1.4_x64-setup.exe` — NSIS 인스톨러 (권장)
- v2.0.x / v2.1.x 사용자는 위젯 Settings → "업데이트 확인" 으로 자동 수신 가능

### macOS (Apple Silicon)
- `Claude Widget_2.1.4_aarch64.dmg` — DMG 인스톨러
- 첫 실행 Gatekeeper 우회: Finder 우클릭 → 열기, 또는 `xattr -d com.apple.quarantine "/Applications/Claude Widget.app"`

## 시스템 요구사항

- Windows 10/11 (Win11 권장 — Mica vibrancy)
- macOS 11 Big Sur 이상 (Apple Silicon)
- WebView2 Runtime (Win10 인스톨러가 자동 설치)
